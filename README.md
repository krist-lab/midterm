Midterm Project

A web development project focusing on semantic HTML, SASS structure, and modular JavaScript. This project is part of the midterm assignment and aims to practice proper folder structure and code organization.

Prerequisites

Git installed

Installation

Clone the repository:

git clone https://github.com/YourUsername/midterm.git

Navigate to the project directory:

cd midterm

Install dependencies (if applicable):

npm install

To start the project, run:

npm start
